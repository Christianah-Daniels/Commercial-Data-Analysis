Heritage Bank Customer Analytics Dashboard
Project Overview

This Power BI dashboard analyses customer demographics, income, tenure, and occupation data to help Heritage Bank better understand its customer base and support data-driven decision-making.

Objective

To provide stakeholders with a clear view of customer characteristics, identify trends across regions and banking segments, and support customer engagement and retention strategies.

Tools Used
Power BI
Power Query
DAX
Excel
Key Features
Customer Overview (Total Customers, Average Age, Income, and Tenure)
Regional Analysis (East, North, South, West)
Customer Segment Analysis (Corporate, Private, Retail)
Demographic and Occupation Insights
Interactive Filters and Visualisations
