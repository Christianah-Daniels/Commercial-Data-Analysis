Heritage Bank Customer Demographics Dashboard
Project Overview

This Power BI dashboard analyses customer demographics, income, tenure, and occupational data for Heritage Bank. The goal was to transform raw customer data into meaningful insights that support customer segmentation, engagement strategies, and business decision-making.

Tools Used

Power BI

Power Query

DAX

Microsoft Excel

Key Features

Customer overview (Total Customers, Average Age, Income, and Tenure)

Regional analysis (East, North, South, West)

Customer segment analysis (Corporate, Private, Retail)

Demographic and occupation insights

Interactive filters and visualisations

Key Insight


The dashboard helps identify customer trends, high-value segments, and engagement opportunities, enabling data-driven decisions and improved customer relationship management.

Outcome

Developed an interactive and user-friendly dashboard that provides stakeholders with clear visibility into customer behaviour and business performance.
